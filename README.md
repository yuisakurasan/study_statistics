# 統計学の学習 (Study of Statistics)

統計学の基礎概念をPythonで実装・検証するための学習用リポジトリです。

## 概要 (Overview)

このプロジェクトは、統計学の理論を数式や文章だけでなく、実際のプログラムコード（Python）として実装することで、より深い理解を得ることを目的としています。

* **Language:** Python 3.x
* **Environment:** Jupyter Notebook (.ipynb)
* **Libraries:**
    * `pandas`: データ処理、度数分布表の作成など
    * `numpy`: 数値計算、数式の及実装
    * `matplotlib`: データの可視化（ヒストグラムなど）
    * `openpyxl`: Excelデータの読み込み
